# @chrisaguilar/stylelint-config

> My personal [stylelint](https://stylelint.io) configuration

## Usage

### Install

`$ npm i -D @chrisaguilar/stylelint-config`

### In Your Stylelint Configuration File

```jsonc
{
    // ...
    "extends": "@chrisaguilar/stylelint-config"
}
```
