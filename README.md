# @chrisaguilar/stylelint-config

## Usage

`$ npm i -D stylelint @chrisaguilar/stylelint-config`

Then, in `<projectRoot>/.stylelintrc`:

```json
{
    "extends": "@chrisaguilar/stylelint-config",
    "rules": { "..." }
}
```

OR

In package.json:

```json
{
    "stylelint": {
        "extends": "@chrisaguilar/stylelint-config"
    }
}
```
